# api-gateway-lite

> Lightweight API gateway with rate limiting, auth middleware, and request logging

## 🚀 Quick Start

```bash
git clone https://github.com/lanzlog/api-gateway-lite.git
cd api-gateway-lite
pip install -r requirements.txt
python src/main.py
```

## 📁 Structure

```
api-gateway-lite/
├── src/           # Source code
├── tests/         # Tests
├── docs/          # Documentation
└── README.md
```

## 🧪 Tests

```bash
python -m pytest tests/ -v
```
